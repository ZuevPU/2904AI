⚠️ Важно помнить

  1. Qwen может ошибаться — всегда перепроверяйте факты
  2. Контекст важен — чем подробнее запрос, тем лучше результат
  3. Итерации нормальны — уточняйте и дорабатывайте ответы
  4. Вы — эксперт — ИИ лишь помощник, решение за вами

🚀 С чего начать? 3 простых шага

  1. Определите задачу

    Что именно вам нужно? План урока, тест, идея для проекта?

  2. Сформулируйте запрос

    Используйте формулу: Роль + Задача + Контекст + Формат

  3. Оцените и доработайте

    Если результат не идеален — уточните запрос

💡 Пример запроса для викторины:

Создай викторину по рассказу "Муму" Тургенева. 10 вопросов с вариантами ответов. Включи вопросы на понимание сюжета и анализ характеров.

📈 Этап 2: От простого к сложному

Qwen поддерживает разные уровни сложности запросов — от быстрых вопросов до создания интерактивных артефактов.

Уровень 1

⚡ Быстрые вопросы

Простые запросы для мгновенных ответов

Пример:
«Объясни тему "Фотосинтез" простыми словами»

Результат для ученика: Краткое понятное объяснение термина или концепции

Уровень 2

📋 Структурированный контент

Таблицы, сценарии, карточки знаний

Пример:
«Создай таблицу сравнения климатических поясов»

Результат: Готовый структурированный материал для урока

Уровень 3

🔀 Ветвящиеся сценарии

Текстовые квесты и интерактивные истории

Пример:
«Создай текстовый квест по теме "Экосистемы"»

Результат: Интерактивный сценарий с выбором действий

🎨 Артефакты

Интерактив в браузере: викторины, тренажёры, таймеры

🎓 Этап 4: Интерактивные уроки с Qwen

🤝 Qwen как со-автор урока

Используйте ИИ для создания полноценных интерактивных уроков. Вот пример запроса:

Ты — профессиональный методист. Создай план интерактивного урока по теме "[ТЕМА]" для [КЛАСС] класса.

Включи:
1. Организационный момент (2 мин)
2. Актуализацию знаний (5 мин) — мини-викторина
3. Изучение нового материала (15 мин) — интерактивная презентация
4. Практическую работу (15 мин) — групповое задание
5. Рефлексию (5 мин) — опрос
6. Домашнее задание (3 мин)

Создай HTML-страницу со всеми интерактивными элементами.
🔗 Демо-пример: Попробуйте создать аналогичный урок для своего предмета!

📝 Расширенный промпт-шаблон

Готовый шаблон для создания комплексных учебных материалов

Ты — профессиональный методист с 20-летним опытом работы в школе. Твоя специализация — создание современных интерактивных уроков с использованием цифровых технологий.

📚 ЗАДАЧА:
Создай подробный план урока по теме "[ВАША ТЕМА]" для учащихся [КЛАСС] класса.

📖 КОНТЕКСТ:
- Продолжительность урока: 45 минут
- Количество учеников: 25 человек
- Технические возможности: проектор, компьютер у учителя, смартфоны у учеников
- Предварительные знания: [ОПИШИТЕ]

🎯 ЦЕЛИ УРОКА:
1. Образовательная: [что должны узнать]
2. Развивающая: [какие навыки развить]
3. Воспитательная: [какие ценности формировать]

📋 СТРУКТУРА УРОКА:

1. ОРГАНИЗАЦИОННЫЙ МОМЕНТ (2 мин)
   - Приветствие
   - Проверка готовности
   - Мотивация через проблемный вопрос

2. АКТУАЛИЗАЦИЯ ЗНАНИЙ (5 мин)
   - Интерактивная викторина (5 вопросов)
   - Обсуждение результатов

3. ИЗУЧЕНИЕ НОВОГО МАТЕРИАЛА (15 мин)
   - Краткая лекция с визуальной поддержкой
   - Работа с интерактивной схемой/таблицей
   - Примеры из реальной жизни

4. ПРАКТИЧЕСКАЯ РАБОТА (15 мин)
   - Групповое задание (4 группы по 6 человек)
   - Каждая группа получает свой кейс
   - Презентация результатов (по 2 мин на группу)

5. РЕФЛЕКСИЯ (5 мин)
   - Метод "Незаконченное предложение"
   - Оценка собственного понимания темы (1-10)

6. ДОМАШНЕЕ ЗАДАНИЕ (3 мин)
   - Базовый уровень (для всех)
   - Продвинутый уровень (по желанию)
   - Творческое задание (на выбор)

🛠️ ИНТЕРАКТИВНЫЕ ЭЛЕМЕНТЫ:
Создай работающие HTML-компоненты:
- Викторина с автоматической проверкой
- Интерактивная таблица/схема
- Таймер для групповой работы
- Форма для рефлексии

💡 ДИФФЕРЕНЦИАЦИЯ:
- Для успевающих: дополнительные вопросы со звёздочкой
- Для испытывающих трудности: карточки-подсказки
- Для визуалов: схемы и инфографика
- Для кинестетиков: практические манипуляции

📊 КРИТЕРИИ ОЦЕНКИ:
- "5" — 90-100% правильных ответов
- "4" — 75-89%
- "3" — 50-74%
- "2" — менее 50%

🎁 БОНУС: "Вау-момент"
Придумай неожиданный факт или демонстрацию, которая запомнится ученикам надолго.

👥 Задание для групповой работы

Практикум по созданию промптов для вашего предмета

  • Шаг 1: Объединитесь в группы по предметам (2-3 человека)
  • Шаг 2: Выберите тему из вашей рабочей программы
  • Шаг 3: Используя конструктор, создайте промпт для этой темы
  • Шаг 4: Протестируйте промпт в Qwen, при необходимости доработайте
