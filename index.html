Этап 1. Просто о возможностях

❗ Важные напоминания

🔒 Не загружайте в чат ФИО учеников, фото, персональные данные
✅ Проверка: всегда адаптируйте и проверяйте материалы перед уроком
🛠 Техника: протестируйте ссылки и артефакты ДО урока
🌿 Гибкость: если интерактив «не зашёл» — это не провал, это обратная связь
🎯 Главное: технология — инструмент, а не цель. В центре — ученик и его развитие.
⚡️ Критическое мышление: учите учеников проверять факты и ставить под вопрос ответы ИИ.

📚 Чем поможет нейросеть учителю?

1. Подготовка к урокам

Что нужно учителю Как поможет нейросеть
План урока Напишет структуру за 1 минуту: цели, этапы, задания
Тесты и вопросы Создаст вопросы разного уровня сложности с ответами
Раздаточные материалы Сгенерирует карточки, памятки, чек-листы
Идеи для активностей Предложит игры, дискуссии, проекты по теме

2. Работа с текстом

  • Адаптация материалов: упростит сложный текст под возраст учеников
  • Проверка работ: найдёт ошибки, даст комментарии по структуре (решение остаётся за учителем)
  • Обратная связь: поможет сформулировать мягкие и конструктивные комментарии
  • Перевод: быстро переведёт материалы с иностранных языков

3. Идеи и творчество

  • Придумает тему для проектной работы
  • Предложит примеры из жизни для объяснения темы
  • Создаст сценарий внеклассного мероприятия
  • Подскажет, как объяснить сложное простыми словами

⚠️ Важно помнить

  • Нейросеть — помощник, а не замена учителю
  • Всегда проверяйте факты и адаптируйте материалы под свой класс
  • Не загружайте персональные данны учеников в чат
  • Итоговое решение и ответственность — за вами

🚀 С чего начать? 3 простых шага

  1. Задайте конкретный запрос:

    "Составь план урока по теме 'Фотосинтез' для 6 класса, 45 минут"

  2. Уточните детали:

    "Добавь интерактивное задание и вопросы для проверки"

  3. Отредактируйте под себя: Возьмите лучшее, добавьте свой опыт и стиль

Пример запроса, который сработает:

"Помоги создать 5 вопросов для викторины по рассказу 'Муму' для 5 класса. Вопросы должны быть разного типа: на внимание, на понимание, на размышление."

Уровень 4: Артефакты нейросети — интерактив в браузере

Артефакты — функция нейросети, которая создаёт готовые мини-приложения: викторины, тренажёры, таймеры, доски идей. Работают в браузере, без установки программ.

Как это работает (очень просто):

  1. 📝 Пишете запрос в нейросеть (как в примерах ниже)
  2. 🧩 Нажимаете кнопку "Artifacts" или "Создать"
  3. 📦 Получаете готовый материал или код
  4. 🌐 Открываете на уроке — всё работает в браузере

Что можно сделать за 1 запрос:

Запрос учителю Что создаст нейросеть Как использовать на уроке
"Создай интерактивную викторину из 5 вопросов по математике для 4 класса: кнопки выбора, проверка ответа, счётчик баллов" Готовая веб-викторина с кнопками и автоматической проверкой Открыть на проекторе — ученики отвечают по очереди
"Сделай тренажёр 'Собери слово': перемешанные буквы, поле для ввода, кнопка 'Проверить'" Интерактивное упражнение на правописание Работа в парах за компьютером или на планшете
"Создай таймер с визуальным отсчётом на 5 минут для групповой работы" Красивый таймер с прогресс-баром Запустить на доске во время активности
"Сделай простую игру 'Угадай страну по флагу': 5 вариантов, подсказки, счёт" Обучающая мини-игра Закрепление темы в игровой форме

📝 Расширенный промпт-шаблон

Ты — профессиональный методист и педагог-дизайнер с опытом создания современных уроков.
ЗАДАЧА:
Создай детальный план гибридного урока (офлайн + цифровые элементы), который будет интересен РАЗНЫМ типам учеников.
ПАРАМЕТРЫ УРОКА:
• Предмет: [Предмет]
• Класс: [Класс]
• Тема: [Тема]
• Длительность: [40 мин]
• Техническое оснащение: [проектор, смартфоны, интерактивная доска, базовый набор...]
УЧТИ РАЗНЫЕ ТИПЫ УЧЕНИКОВ:
✓ Визуалы → схемы, инфографика, видео, цветовое кодирование
✓ Аудиалы → обсуждения, подкасты, озвучка, дебаты
✓ Кинестетики → движение, манипуляции, эксперименты, ролевые сценки
✓ Цифровые аборигены → гаджеты, интерактив, геймификация, мгновенная обратная связь
✓ Медлительные/тревожные → чёткие инструкции, время на осмысление, безопасная среда
✓ Одарённые/быстрые → дополнительные челленджи, исследовательские мини-задачи
СТРУКТУРА ОТВЕТА (строго по пунктам):
ЦЕЛИ (по таксономии Блума)
- Знать/Понимать/Применять/Анализировать...
ПОЭТАПНЫЙ ПЛАН (с таймингом)
Для каждого этапа укажи:
1. Название этапа + время
2. Действия учителя (конкретные фразы, инструкции)
3. Действия учеников
4. [Интерактив] — какой элемент и для каких типов учеников
5. [Цифровой инструмент] — название сервиса/приложения + как использовать
6. [Адаптация] — как включить всех типов учеников
БАНК ИНТЕРАКТИВНЫХ ПРИЁМОВ (выбери 4-6, подходящих под тему)
□ Быстрый квиз (Kahoot/Quizizz/Google Forms)
□ Ментальная карта в реальном времени (Miro/Jamboard)
□ Голосование с визуализацией (Mentimeter)
□ "Живая статистика" — ученики перемещаются по классу
□ Микро-исследования в парах с гаджетами
□ Создание контента: мемы, тикток-сценарии, комиксы по теме
□ Ролевая мини-игра / дебаты / "суд над понятием"
□ Квест с QR-кодами по классу/школе
□ Интерактивное видео с остановками и вопросами
□ Коллаборативный документ (класс заполняет вместе)
МАТЕРИАЛЫ И РЕСУРСЫ
• Для учителя: чек-лист подготовки, скрипты фраз, критерии наблюдения
• Для учеников: раздатка (текст/схема/чек-лист), ссылки, QR-коды
• Цифровые инструменты: названия + краткая инструкция "как за 1 минуту"
• Запасной офлайн-план: если техника подведёт
СЕКРЕТЫ ВОВЛЕЧЕНИЯ (практические лайфхаки)
→ Как переключить внимание, если класс "потерялся"
→ Как дать слово тихим ученикам без стресса
→ Как превратить ошибку в учебный момент
→ Как быстро проверить понимание ВСЕХ (не только активных)
ОЦЕНИВАНИЕ
• Формирующее: 2-3 быстрых способа "снять пульс" урока
• Критерии успеха: что ученик сможет СДЕЛАТЬ после урока
• Самооценка: простой шаблон для рефлексии ученика
• Домашнее задание: 3 варианта на выбор (базовый / творческий / исследовательский)
БОНУС: "Вау-момент"
Предложи 1 неожиданный элемент, который запомнится:
- Связь с реальной жизнью / профессиями
- Неочевидный факт или парадокс по теме
- Мини-эксперимент, который можно сделать прямо сейчас
- Эмоциональный якорь (история, образ, метафора)
СТИЛЬ ОТВЕТА:
• Конкретно, без воды — учитель должен взять и провести
• Примеры фраз, названий сервисов, временных рамок
• Визуальное оформление: эмодзи, маркеры, чёткие заголовки
• Адаптивность: пометки [для начальной школы] / [для старшеклассников], где уместно
ВАЖНО:
Не предлагай интерактив ради интерактива. Каждый элемент должен работать на цель урока и учитывать разнообразие учеников.

Инструкция: как создать приложение

🔗 Открыть QWEN
  • Выберите режим: «Артефакты»
  • Добавьте задачу: прикрепите файл при необходимости
  • Введите промпт и дождитесь генерации — ИИ создаст HTML/CSS/JS код
  • Нажмите «Предварительный просмотр» или «Запустить», чтобы увидеть приложение прямо в чате
  • Протестируйте приложение: проверьте работу кнопок и полей ввода, убедитесь, что логика верна — математическая и алгоритмическая

  • Нажмите «Развернуть», чтобы открыть приложение на весь экран без интерфейса чата
  • Скопируйте ссылку из адресной строки или через вкладку «Поделиться»
